# 9.1 基础指令表

## 9.1.1 运动类

### MOVJ
*   **功能**：关节运动指令
*   **格式**：
    ```text
    MOVJ P[值]\PR[值] V = <参数2> ACC = <参数3> CNT = <参数4> \ FINE
    ```
*   **说明**：
    *   参数1：P[值]\PR[值] 位置变量号。
    *   参数2：关节运动的运行速度，单位%，范围1-100。
    *   参数3：最大加速度百分比，单位：%，范围1-120。
    *   参数4：CNT :轨迹连续的逼近等级，可输入0-100。 \ FINE:关节运动精准到达。
*   **举例**：
    ```text
    MOVJ P V = 100 ACC = 20 CNT = 1
    ```

### MOVL
*   **功能**：直线运动指令
*   **格式**：
    ```text
    MOVL P[值]\PR[值] V = <参数2> ACC = <参数3> CNT = <参数4> \ FINE
    ```
*   **说明**：
    *   参数1：P[值]\PR[值] 位置变量号。
    *   参数2：直线运动的运行速度，单位mm/s，范围0.1-2500。
    *   参数3：最大加速度百分比，单位：%，范围1-120。
    *   参数4：CNT :轨迹连续的逼近等级，可输入0-100。 \ FINE:关节运动精准到达。
*   **举例**：
    ```text
    MOVL P V = 100 ACC = 20 CNT = 1
    ```

### MOVC
*   **功能**：圆弧运动指令
*   **格式**：
    ```text
    MOVL P[值]\PR[值] V = <参数2> ACC = <参数3> CNT = <参数4> \ FINE
    ```
*   **说明**：
    *   参数1：P[值]\PR[值] 位置变量号。
    *   参数2：圆弧运动的运行速度，单位mm/s，范围0.1-2500。
    *   参数3：最大加速度百分比，单位：%，范围1-120。
    *   参数4：CNT :轨迹连续的逼近等级，可输入0-100。 \ FINE:关节运动精准到达。
*   **举例**：
    ```text
    MOVL P V = 100 ACC = 20 CNT = 1
    ```

## 9.1.2 基础指令

### SET
*   **功能**：赋值指令
*   **格式**：
    ```text
    SET  <参数1>  <参数2>
    ```
*   **说明**：
    *   参数1：I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / P[]：位姿变量，能直接赋值 / S[]：字符型变量，能直接赋值 / P[].X P[].Y P[].Z / P[].roll P[].pitch P[].yaw / P[].j1 P[].j2 P[].j3 / P[].j4 P[].j5 P[].j6 / P[].e1 P[].e2 P[].e3 / P[].e4 P[].e5 P[].e6
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / P[]：位姿变量，能直接赋值 / S[]：字符型变量，能直接赋值 / P[].X P[].Y P[].Z / P[].roll P[].pitch P[].yaw / P[].j1 P[].j2 P[].j3 / P[].j4 P[].j5 P[].j6 / P[].e1 P[].e2 P[].e3 / P[].e4 P[].e5 P[].e6
*   **注**：P指令只能与P做赋值运算。
*   **举例**：
    ```text
    SET I I
    SET R[I] 5
    ```

### SETP (修改关节配置)
*   **功能**：修改P[]\PR[]变量的MWORD值或坐标系号，修改mword前三个值，代表关节配置。
*   **格式**：
    ```text
    SETP  <参数1>  <参数2>
    ```
*   **说明**：
    *   参数1：P[值]/PR[值]
    *   参数2：CFG ：关节配置，三个字母分别代表5、3、1轴的控制点位于控制面的哪一侧。
    *   参数3：F U T / N U T / F D T / F U B / N D T / F D B / N U B / N D B
    *   注释：1、FLIP\NOFLIP,代表手腕的上下；2、UP\DOWN,代表手臂的上下；3、BACK\FRONT代表手臂的前后。
*   **举例**：
    ```text
    SETP P CFG FUT
    ```

### SETP (修改回转数)
*   **功能**：修改P[]\PR[]变量的MWORD值或坐标系号，修改mword后三个值，代表回转数
*   **格式**：
    ```text
    SETP  <参数1>  <参数2>
    ```
*   **说明**：
    *   参数1：P[值]/PR[值]
    *   参数2：TN ：回转数
    *   参数3：1:1 J1轴 / 2:4 J4轴 / 3:6 J6轴
    *   参数4：-1：关节在-540°到-180°之间取值。 / 0：关节在-180°到180°区间取值。 / 1：关节在180°到540°之间取值。
*   **举例**：
    ```text
    SETP PR TN 1 1
    ```

### STEP (修改坐标系号)
*   **功能**：修改P[]\PR[]变量的MWORD值或这坐标系号，修改坐标系号
*   **格式**：
    ```text
    SETP  <参数1>  <参数2>
    ```
*   **说明**：
    *   参数1：P[值]/PR[值]
    *   参数2：FRAME
    *   参数3：TF 工具坐标系号 / UF 用户坐标系号 / ET 外部工具坐标系号
    *   参数4：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
*   **举例**：
    ```text
    SETP P FRAME TF 1
    ```

### SETFRAME
*   **功能**：切换坐标系指令。
*   **格式**：
    ```text
    SETFRAME <参数1>
    ```
*   **说明**：
    *   参数1：TF 工具坐标系号（1-8）
    *   参数2：UF 用户坐标系号（1-8）
    *   参数3：ET 外部工具坐标系号（1-8）
*   **举例**：
    ```text
    SETFRAME TF 1
    ```

### GETP
*   **功能**：获取当前位置信息，保存到P[值]/PR[值]变量中。
*   **格式**：
    ```text
    GETP <参数1>
    ```
*   **说明**：
    *   参数1：P[常数]/PR[常数]
*   **举例**：
    ```text
    GETP PR
    ```

### SETEXTAXIS
*   **功能**：对一个位置变量P[]/PR[]中的一个外部轴进行赋值。
*   **格式**：
    ```text
    SETEXTAXIS  <参数1>  <参数2>  <参数3>
    ```
*   **说明**：
    *   参数1：P[值]/PR[值]
    *   参数2：e[]
    *   参数3：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量
*   **注**：通过此指令变更外部轴关节值，保证tcp不变得情况下，机器人会发生轨迹变化。
*   **举例**：
    ```text
    SETEXTAXIS P e1 50
    ```

### CLEAR
*   **功能**：清空指定范围的变量值。
*   **格式**：
    ```text
    CLEAR  <参数1>  <参数2>
    ```
*   **说明**：
    *   参数1：I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量
    *   参数2：I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量
*   **注**：此指令参数1与参数2必须为同类型变量
*   **举例**：
    ```text
    CLEAR I I
    CLEAR B[I] B[I]
    ```

### STRLEN
*   **功能**：将参数1的长度值，存入参数2中。
*   **格式**：
    ```text
    STRLEN  <参数1>  <参数2>
    ```
*   **说明**：
    *   参数1：S[常数]/S[I[常数]] (直接/间接) / S[]:字符型变量，能直接赋值
    *   参数2：I[常数]/I[I[常数]] (直接/间接) / I[]：32位整型变量
*   **举例**：
    ```text
    STRLEN S I
    STRLEN S[I] I[I]
    ```

### STRFIND
*   **功能**：查找部分字符串，从参数1中查找参数2出现的索引值，结果放参数3中。
*   **格式**：
    ```text
    STRFIND  <参数1> <参数2> <参数3>
    ```
*   **说明**：
    *   参数1：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
    *   参数2：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
    *   参数3：I[常数]/I[I[常数]] (直接/间接) / I：32位整型变量
*   **举例**：
    ```text
    STRFIND S S I
    STRFIND S[I] S[I] I[I]
    ```

### STRSUB
*   **功能**：获取部分字符串，从参数1中获取参数2与参数3长度的字符串，结果放参数4中。
*   **格式**：
    ```text
    STRSUB <参数1> <参数2> <参数3> <参数4>
    ```
*   **说明**：
    *   参数1：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
    *   参数2：I[常数]/I[I[常数]] (直接/间接) / I：32位整型变量
    *   参数3：I[常数]/I[I[常数]] (直接/间接) / I：32位整型变量
    *   参数4：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
*   **举例**：
    ```text
    STRSUB S I I S
    STRSUB S[I] I[I] I[I] S[I
    ```

### STRCMP
*   **功能**：字符串比较，从参数1与参数2进行比较，结果放参数3。比较结果相同为1，不同为0。
*   **格式**：
    ```text
    STRCMP <参数1> <参数2> <参数3>
    ```
*   **说明**：
    *   参数1：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
    *   参数2：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
    *   参数3：I[常数]/I[I[常数]] (直接/间接) / I：32位整型变量
*   **举例**：
    ```text
    STRCMP S S I
    STRCMP S[I] S[I] I[I]
    ```

### GETAT
*   **功能**：获取字符串单个字符，将参数1中索引为参数2的字符放到参数3中。
*   **格式**：
    ```text
    STRCMP <参数1> <参数2> <参数3>
    ```
*   **说明**：
    *   参数1：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
    *   参数2：I[常数]/I[I[常数]] (直接/间接) / I：32位整型变量
    *   参数3：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
*   **举例**：
    ```text
    GETAT S I S
    GETAT S[I] I[I] S[I]
    ```

### STRFINDEND
*   **功能**：字符串逆向查找，以逆序从参数1中查找参数2，并将参数2出现的索引存入参数3
*   **格式**：

### STRSPLIT
*   **功能**：字符串分割功能，将参数1为源字符串，以参数2为分隔符分割，分割结果为一个字符串数组，存储在参数3为起始索引的若干个局部字符串变量中参数4为分割后的字符串数量
*   **格式**：
    ```text
    STRSPLIT <参数1> <参数2> <参数3> <参数4>
    ```
*   **说明**：
    *   参数1：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
    *   参数2：S[常数]/S[I[常数]] (直接/间接) / S:字符型变量，能直接赋值
    *   参数3：常数
    *   参数4：I[常数]/I[I[常数]] (直接/间接) / I：32位整型变量
*   **举例**：
    ```text
    STRSPLIT S S 0 I
    STRSPLIT S[I] S[I] 0 1[I]
    ```

## 9.1.3 运算指令

### COP
*   **功能**：运算类符合指令，支持各种运算的组合，组合没有长度限制。
*   **格式**：
    ```text
    <参数1>  =  <参数2>  [运算符] <参数3> …
    ```
*   **说明**：
    *   参数1：I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / S[]:字符型变量，能直接赋值 / P[]：位姿变量，能直接赋值 / P[].X P[].Y P[].Z / P[].RX P[].RY P[].RZ / P[].j1 P[].j2 P[].j3 / P[].j4 P[].j5 P[].j6 / P[].e1 P[].e2 P[].e3 / P[].e4 P[].e5 P[].e6
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / P[]：位姿变量，能直接赋值 / P[].X P[].Y P[].Z / P[].RX P[].RY P[].RZ / P[].j1 P[].j2 P[].j3 / P[].j4 P[].j5 P[].j6 / P[].e1 P[].e2 P[].e3 / P[].e4 P[].e5 P[].e6
    *   运算符：+  -  *  /  >>  <<  &  |  &&  || ^
*   **注释**：
    1. 针对P.1轴关节值的修改会修改整个P的姿态值。
    2. P只支持+运算即实现P的增量变换（矩阵运算），如p = p + p。
*   **运算类型与变量类型**：
    *   ADD：常数I H R B S P P.
    *   SUB：常数I H R B P.
    *   MUL：常数I H R B P.
    *   DIV：常数I H R B P.
    *   AND：常数I H B
    *   OR：常数I H B
    *   RIGHTSHIFT：常数I H B
    *   LEFTSHIFT：常数I H B
*   **举例**：
    ```text
    I = I + I
    ```

### NOT
*   **功能**：按位取反运算。
*   **格式**：
    ```text
    <参数1> = NOT <参数2>
    ```
*   **说明**：
    *   参数1：I：32位整型变量 / H：16位整型变量 / B：8位整型变量
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量
*   **举例**：
    ```text
    I = NOT I
    ```

*   **格式**：
    ```text

    ```
*   **说明**：
    *   参数1：R[]：64位浮点型变量
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / P[].X P[].Y P[].Z / P[].RX P[].RY P[].RZ / P[].j1 P[].j2 P[].j3 / P[].j4 P[].j5 P[].j6 / P[].e1 P[].e2 P[].e3 / P[].e4 P[].e5 P[].e6
*   **举例**：
    ```text
    R = ATAN R
    ```

### TAN
*   **功能**：求参数2的正切值，结果保存到参数1中。
*   **格式**：
    ```text

    ```
*   **说明**：

    ```text

    ```

### SIN
*   **功能**：对参数2进行正弦计算，将得到的结果存入到参数1。
*   **格式**：
    ```text
    R[值] = SIN <参数2>
    ```

## 9.1.4 I/O指令

### IN
*   **功能**：将I/O赋给变量I[值]
*   **格式**：
    ```text
    IN I[值] = <参数2>
    ```
*   **说明**：
    *   参数1：I[]：32位整型变量
    *   参数2：DI[值] / GIB ：8位组IO / GIH ：16位组IO / GII ：32位组IO / GIF ：32位组IO / DO[值] / GOB ：8位组IO / GOH ：16位组IO / GOI ：32位组IO / GOF ：32位组IO
*   **举例**：
    ```text
    IN I[值] = [变量类型][值]
    ```

### OUT
*   **功能**：对I/O和组I/O进行输出。
*   **格式**：
    ```text
    OUT <参数1> = <参数2>
    ```
*   **说明**：
    *   参数1：DO[值] / GOB ：8位组IO / GOH ：16位组IO / GOI ：32位组IO / GOF ：32位组IO
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
*   **举例**：
    ```text
    OUT DO = 1
    OUT GOB = I
    ```

### OUT_T
*   **功能**：当到达参数3指定的时间或距离后，输出I/O指令。
*   **格式**：
    ```text
    OUT <参数1> = <参数2> <参数3> <参数4>
    ```
*   **说明**：
    *   参数1：DO[值] / GOB ：8位组IO / GOH ：16位组IO / GOI ：32位组IO / GOF ：32位组IO
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
    *   参数3：T(单位s) / DIS（单位mm）
    *   参数4：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
*   **注释**：T时间参数支持正、负，负数时，指令仅可作为附加指令存在，-T表示当前剩余时间开始输出。
*   **举例**：
    ```text
    OUT DO = 1 T = -1
    ```

### ANIN
*   **功能**：模拟量输出信号，是连续的信号，一般表示电压、电流等数据。将模拟量输入I/O赋值给I变量。
*   **格式**：
    ```text
    ANIN <参数1> = <参数2>
    ```
*   **说明**：
    *   参数1：R[]：64位浮点型变量
    *   参数2：AI[值]
*   **举例**：
    ```text
    ANIN R[值] = AI [值]
    ```

### ANOUT
*   **功能**：模拟量输出信号，是连续的信号，一般表示电压、电流等数据。将变量值转换成模拟量输出到指定的I/0中。

*   **说明**：
    *   参数1：AO[值]
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / P[].X P[].Y P[].Z / P[].RX P[].RY P[].RZ / P[].j1 P[].j2 P[].j3 / P[].j4 P[].j5 P[].j6 / P[].e1 P[].e2 P[].e3 / P[].e4 P[].e5 P[].e6


## 9.1.5 控制类

### IF OP
*   **功能**：条件判断指令，对条件进行判断，并根据结果进行后续指令的执行。
*   **格式**：
    ```text
    IF <参数1> 操作符 <参数2> <附加指令>
    ```
*   **说明**：
    *   参数1：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / P[].X P[].Y P[].Z / P[].RX P[].RY P[].RZ / DI[值] / GIB ：8位组IO / GIH ：16位组IO / GII ：32位组IO / GIF ：32位组IO / DO[值] / GOB ：8位组IO / GOH ：16位组IO / GOI ：32位组IO / GOF ：32位组IO
    *   操作符：> / < / >= / <= / == / !=
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / P[].X P[].Y P[].Z / P[].RX P[].RY P[].RZ / DI[值] / GIB ：8位组IO / GIH ：16位组IO / GII ：32位组IO / GIF ：32位组IO / DO[值] / GOB ：8位组IO / GOH ：16位组IO / GOI ：32位组IO / GOF ：32位组IO
    *   附加指令：GOTO : 跳转 | CALL : 调用作业 | OUT_T : 到达指定时间、距离后输出I/O | OUT : 输出IO | DELAY : 等待时间 | WAIT : 等待信号
*   **举例**：
    ```text
    IF I < 14
    DELAY T = 1
    ELSEIF I < 10
    DELAY T = 5
    ENDIF
    ```

### IF OP AND\OR OP …
*   **功能**：条件判断指令，对条件进行判断，并根据结果进行后续指令的执行。在条件判断中，加入逻辑运算。
*   **格式**：
    ```text
    IF <参数1> 操作符 <参数2> AND <参数1> 操作符 <参数2> <附加指令>
    ```
*   **说明**：
    *   参数1/参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / P[].X P[].Y P[].Z / P[].RX P[].RY P[].RZ / DI[值] / GIB ：8位组IO / GIH ：16位组IO / GII ：32位组IO / GIF ：32位组IO / DO[值] / GOB ：8位组IO / GOH ：16位组IO / GOI ：32位组IO / GOF ：32位组IO
    *   操作符：> / < / >= / <= / == / !=
    *   逻辑符：AND ：与 / OR ： 或
    *   附加指令：GOTO ：跳转 | CALL ：调用作业 | OUT_T ：到达指定时间、距离后输出I/O | OUT ：输出IO | DELAY ：等待时间 | WAIT ：等待信号
*   **举例**：
    ```text
    IF I < I AND I < I
    ```

### ELSEIF OP
*   **功能**：满足条件执行区间内指令。
*   **格式**：
    ```text
    ELSEIF <参数1> 操作符 <参数2> AND <参数1> 操作符 <参数2> <附加指令>
    ```
*   **说明**：参数支持同上
*   **举例**：
    ```text
    ELSEIF I < I AND I < I : GOTO “L0”
    ```

### ELSE
*   **功能**：与IF指令一起使用，IF条件未满足，执行区间内指令。
*   **格式**：
    ```text
    ELSE
    ```
*   **说明**：参数无
*   **举例**：
    ```text
    IF I < I
    …
    ELSE
    …
    ```

### ENDIF
*   **功能**：与IF指令一起使用，条件跳转指令结束。
*   **格式**：
    ```text
    ENDIF
    ```
*   **说明**：参数无
*   **举例**：
    ```text
    IF I < I
    …
    ELSE
    …
    ENDIF
    ```

### GOTO (IF指令的附加指令)
*   **功能**：跳转到标签行。
*   **格式**：
    ```text
    IF I < I : GOTO “参数1”
    ```
*   **说明**：参数1：标签值
*   **举例**：
    ```text
    IF I < I : GOTO “L0”
    ```

### CALL (IF指令的附加指令)
*   **功能**：调用作业。
*   **格式**：
    ```text
    IF I < I : CALL <参数1>
    ```
*   **说明**：参数1：作业名
*   **举例**：
    ```text
    IF I < I : CALL zuoye.txt
    ```

### OUT_T (IF指令的附加指令)
*   **功能**：到达指定时间、距离后输出I/O
*   **格式**：
    ```text
    IF I < I : OUT <参数1> <参数2> <参数3> = <参数4>
    ```
*   **说明**：同基础的 OUT_T 参数说明
*   **举例**：
    ```text
    IF I < I : OUT DO 1 T = 0
    ```

### OUT (IF指令的附加指令)
*   **功能**：输出IO
*   **格式**：
    ```text
    IF I < I : OUT <参数1> <参数2>
    ```
*   **说明**：同基础的 OUT 参数说明
*   **举例**：
    ```text
    IF I < I : OUT DO 1
    ```

### DELAY (IF指令的附加指令)
*   **功能**：等待一段时间。
*   **格式**：
    ```text
    IF I < I : DELAY T = <参数1>
    ```
*   **说明**：参数1：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量
*   **举例**：
    ```text
    IF I < I : OUT DO 1
    ```

### WAIT (IF指令的附加指令)
*   **功能**：等待指令，等待一个信号一段时间，到时间后，执行后续指令。T = - 1是一直等待。
*   **格式**：
    ```text
    IF I < I : WAIT <参数1> = <参数2> T = <参数3>
    ```
*   **说明**：同基础的 WAIT 参数说明
*   **举例**：
    ```text
    IF I < I : WAIT DI = 1 T = 10
    ```

### WHILE N指令
*   **功能**：与ENDWHILE指令一起使用，在循环区间内，执行N次。
*   **格式**：
    ```text
    ANOUT <参数1> = <参数2>
    ```
*   **说明**：参数1：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
*   **举例**：
    ```text
    WHILE N
    ……
    ENDWHILE
    ```

### WHILE OP指令
*   **功能**：条件满足执行区间指令结束。
*   **格式**：
    ```text
    WHILE <参数1> 操作符 <参数2>
    ```
*   **说明**：同 IF OP 参数支持
*   **举例**：
    ```text
    WHILE I>I
    …
    ENDWHILE
    ```

### LABEL
*   **功能**：标签指令
*   **格式**：
    ```text
    LABEL <参数1>
    ```
*   **说明**：参数1：S：字符型变量，能直接赋值
*   **举例**：
    ```text
    LABER “L10”
    ```

### GOTO
*   **功能**：跳转指令，跳转至标签位置。
*   **格式**：
    ```text
    GOTO <参数1>
    ```
*   **说明**：参数1：S：字符型变量，能直接赋值
*   **举例**：

### SKIP CONDITON
*   **功能**：跳过指令。与运动指令的附加指令SKIP“S”一起使用，当满足指令条件时，跳过带有SKIP附加指令的运动指令。
*   **格式**：
    ```text
    SKIP CONDITON <参数1> == <参数2>
    ```
*   **说明**：
    *   参数1：DI[值] / GIB ：8位组IO / GIH ：16位组IO / GII ：32位组IO / GIF ：32位组IO / DO[值] / GOB ：8位组IO / GOH ：16位组IO / GOI ：32位组IO / GOF ：32位组IO
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
*   **举例**：
    ```text
    SKIP CONDITON DI == 1
    MOVJ P V=20 ACC=20 CNT=1 : SKIP”1”
    MONJ P V=20 ACC=20 CNT=1
    ```

### CALL
*   **功能**：调用子作业指令。
*   **格式**：
    ```text
    CALL <参数1>
    ```
*   **说明**：参数1：S：字符型变量，能直接赋值
*   **注**：作业名称需要附带.txt后缀；CALL不能调用自身；如果系统除当前正在编辑的作业外无其他作业，此时插入CALL指令时，参数默认为null.txt作为占位，否则会按照字典顺序排序的第一个作业名做为参数。
*   **举例**：
    ```text
    GOTO 123.txt
    ```

### WAIT
*   **功能**：等待指令，等待一个信号一段时间，到时间后，执行后续指令。T = -1是一直等待一段时间。
*   **格式**：
    ```text
    WAIT <参数1>  = <参数2>  T = <参数3>
    ```
*   **说明**：同上
*   **举例**：
    ```text
    WAIT DI = 1 T = 10
    ```

### DELAY
*   **功能**：延时等待一段时间。
*   **格式**：
    ```text
    DELAY T = <参数1>
    ```
*   **说明**：参数1：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量
*   **举例**：
    ```text
    DELAY T = 1
    ```

### START BG

*   **举例**：

### FOR...ENDFOR
*   **功能**：FOR循环指令。
*   **格式**：
    ```text
    FOR<参数1>=<参数2><参数3><参数4><参数5><参数6>
    ```
*   **说明**：
    *   参数1：I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
    *   参数2：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
    *   参数3：I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
    *   参数4：> / < / > = / < =
    *   参数5：常数 / I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量
    *   参数6：常数
*   **ENDFOR说明**：与FOR 指令组合使用，表示 FOR指令执行完成。
*   **注**：参数3需要与参数1变量类型相同，ENDFOR指令与FOR配对出现。
*   **举例**：
    ```text
    FOR I = 0 I < 4 STEP 1
    ENDFOR
    ```

### SWITCH…CASE…DEFAULT…ENDSWITCH
*   **功能**：条件跳转指令，当某一个条件满足时，将程序跳转到指定的地方运行。
*   **格式**：
    ```text
    SWITCH <参数1>
    ```
*   **说明**：
    *   SWITCH 参数1：I[]：32位整型变量 / H[]：16位整型变量 / B[]：8位整型变量 / R[]：64位浮点型变量 / S[]：字符型变量，能直接赋值
    *   CASE 格式：`CASE<参数1>`
    *   CASE 说明：与SWITCH 指令组合使用，表示 SWITCH ... CASE...指令执行完成。
    *   DEFAULT 说明：与SWITCH ... CASE...在一起使用，当CASE条件没有满足，则执行DEFAULT。
    *   ENDSWITCH 说明：与SWITCH ... CASE...DEFAULT在一起使用，代表程序结束。
*   **注**：DEFAULT、ENDSWITCH指令必须存在。
*   **举例**：
    ```text
    SWITCH I
    CASE I
    BREAK
    CASE I
    BREAK
    DEFAULT
    BREAK
    ENDSWITCH
    ```

*   **举例**：
